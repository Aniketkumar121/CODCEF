# FeaNet: Feedback Cascaded Network for Camouflaged Object Detection


> Authors:
> [kaihong huang](kaihong.huang@seu.edu.cn)


## Prerequisites
``` python
torch==1.3.1
scipy==1.2.2
torchvision==0.4.2
opencv-python==3.4.2.17
```

## Usage
```bash
git clone https://github.com/HHHKKKHHH/FeaNet
cd FeaNet/
pip install -r requirements.txt
```
### Train
```bash
python train.py
```
### Test
```bash
python test.py
```
